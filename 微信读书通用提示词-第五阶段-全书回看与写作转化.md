# 微信读书通用提示词，第五阶段，全书回看与写作转化

只有当我明确说“第五阶段”“全书回看与写作转化”“核心问题收束”“把全书材料转成写作方向”之类的话，且整本书已经读完、第四阶段已经完成后，才执行这一阶段。

第五阶段的标准模型是 **核心问题压缩 + 证据-反证表 + 张力地图 + 五异法细读点 + 写作转化 + 作家写法档案 + 跨作品联动**。它要做的，不是替我把文章一口气写完，而是把全书材料压到能继续写、能被反证、能被文本推进的状态。

下文中的 `{中间稿路径}` 与 `{正式笔记路径}`，分别指总提示词或当前书目专用 router 里固定的该书中间整理稿路径与正式阅读笔记路径。

## 1. 第四阶段与第五阶段的边界

- 第四阶段处理的是，把整本书读完后的卡片、修订链、待回看项和文章方向，收束成 **阅读现场档案 + 文章素材索引**。
- 第五阶段处理的是，在第四阶段已经形成整书结构之后，把这些材料继续压缩成 **核心问题、证据与反证、张力主次、细读入口和写作路径**。
- 第四阶段的重心是整书归档、结构补齐和阅读轨迹保留。第五阶段的重心是问题压缩、反证修正和写作转化。
- 第五阶段不是补做缺失卡片的整理工，不是把第四阶段的文章方向换个说法重抄一遍，也不是把所有材料直接拼成一篇完成稿。
- 若第四阶段结果里还没有可回链的卡片锚点、阅读轨迹或文章方向，只剩空泛主题词和总评句，本阶段不能硬上，必须先回到第四阶段补结构。

## 2. 进入条件与阶段定位

进入第五阶段前，至少确认以下四点：

1. 整本书已经读完，不再处于按章节继续追加材料的状态。
2. `{正式笔记路径}` 中已经形成第四阶段的全书收束结果，至少能看到 `阅读现场档案`、`文章素材索引`、`阅读轨迹与判断变化` 或与之等价的整书结构。
3. 当前书目的主要判断，都还能回到具体卡片、场景、动作、物件、称呼、空间或叙事安排，不是只剩抽象口号。
4. 当前任务目标，是把材料推进到写作准备，不是继续做资料录入、局部润色或迁移归档。

第五阶段的定位，是把全书材料继续压缩成一套可以往下写的工作结构。这里的“写”，首先是短札、短文、长文提纲等写作准备，不等于立刻交付成文终稿。

## 3. 先过计划门，再进入执行

第五阶段默认按复杂任务处理。执行顺序如下：

1. 先读根 `AGENTS.md`、总提示词、第四阶段提示词、当前书目的正式阅读笔记和必要的中间稿。
2. 先起草第五阶段计划，写清当前要压缩的核心问题簇、预期产物、验证方式和范围边界。
3. 起草后交给 Momus、Metis、Oracle 审查。
4. 吸收审查意见，形成审查后的执行计划，再呈现给用户。
5. 只有用户明确授权执行，才进入第五阶段产出。

若当前任务已经触发计划门，计划工件统一遵守以下 canonical block：

> **计划工件默认规则**：当当前任务已经触发复杂任务计划门、阶段计划门、提示词修改计划门，或用户 / 当前适用提示词明确要求“建 plan / 起草计划 / 制定详细计划 / 先过计划门”时，AI 应直接创建或更新 `.omo/plans/` 下的计划工件，不需要额外再问一次“是否允许写 `.omo/plans`”。这属于计划阶段本身，不等于执行产品文件修改。

## 4. 第五阶段不是什么

进入第五阶段后，先明确这些边界：

1. 不是继续整理微信读书划线。
2. 不是继续补卡或把所有轻卡强行扩成完整卡。
3. 不是重跑第三阶段的迁移。
4. 不是重跑第四阶段的全书收束整合。
5. 不是直接写成长文。
6. 不是把阅读笔记改成学术论文。
7. 不是把主题词、立场词、价值判断口号冒充成核心问题。

如果当前材料还停在“我大概想写某个主题”，却没有可回链的文本支点、反证、张力主次和细读入口，说明还没真正进入第五阶段。

## 5. 第五阶段标准产物骨架

第五阶段完成后，默认至少应形成以下骨架。它可以写入用户指定的第五阶段承接位置，也可以先以结构化工作稿交付，但不能把这些模块省掉。

```markdown
## 全书回看与写作转化

### 一、核心问题压缩

### 二、证据-反证表

### 三、张力地图

### 四、五异法细读点

### 五、写作转化

### 六、作家写法档案

### 七、跨作品联动
```

这七部分的关系是：先压问题，再补证据与反证，再压张力，再选细读入口，最后转入写作方向与作家写法档案。不要倒过来先想标题、先写文章，再回头拼证据。

## 6. 核心问题压缩

第五阶段首先要做的，是把第四阶段留下的材料簇压成 3 至 5 个 **核心问题**。通常优先压成 3 个，但若一本书确实需要 4 个或 5 个，允许保留，不为了整齐硬砍。

每个核心问题至少包含以下字段：

| 字段 | 要求 |
|---|---|
| 问题来源 | 这个问题从哪些卡片、场景、反复出现的矛盾里压出来 |
| 为什么值得成为核心问题 | 它为什么能统摄多张卡，为什么不是局部感叹 |
| 问题阶梯 | 按 1 星到 5 星问题阶梯，说明它怎样从情节追问推进到写法或主题追问 |
| 最终推荐表达 | 最终应落成一个可被反驳、可被推进的 5 星问题句，不写成主题词 |
| 关联证据 | 对应哪些卡片、场景、动作、物件、称呼、空间或叙事安排 |
| 可写方向 | 这个问题更适合写短札、主题短文，还是长文主轴 |

核心问题的写法要求：

1. 问题必须能被文本推进，也能被反证修正。
2. 问题不能只写成“城乡差距”“尊严”“爱情”“成长”这类主题词。
3. 问题不能只是道德裁判句，例如“某人是不是坏人”。
4. 若一个问题目前只有立场、没有证据链，先降回候选问题，不要硬抬成核心问题。
5. 若两个问题只是同义改写，保留更能统摄材料、也更能进入写作的一条。

## 7. 证据-反证表

每个核心问题都必须配一张 **证据-反证表**。只列支持证据，不列反证，会让第五阶段重新滑回口号化判断。

每个核心问题至少要写清：

| 字段 | 要求 |
|---|---|
| 支持证据 | 哪些文本细节支撑当前判断 |
| 反证 | 哪些文本会顶撞、拖慢、复杂化当前判断 |
| 判断容易跳跃处 | 目前最可能偷步的地方在哪里 |
| 还需回看的文本 | 哪些段落、场景、关系链还需要补证 |
| 反证如何修正判断 | 反证进入后，原判断需要缩窄、改写还是反向推进 |
| 可发展文章方向 | 这张表最适合发展成什么类型的写作 |

这里的反证，不等于把原判断推翻到底，也不等于为了平衡而机械写一条“另一面”。反证的作用，是逼判断更贴文本、更能承受复杂性。

## 8. 张力地图压缩

第五阶段的 **张力地图**，不是把对立词一条条罗列出来，而是先列全书张力，再压出主次结构，最后看哪些张力真的值得进入写作主轴。

张力地图至少包括以下四层：

1. 全书主张力。
2. 人物心理张力。
3. 关系伦理张力。
4. 写法或结构张力。

每条张力都必须落到具体文本单位，优先落到人物、场景、动作、物件、称呼、空间或叙事安排。不要只写抽象对立词。

建议字段如下：

| 字段 | 要求 |
|---|---|
| 张力层级 | 主张力 / 次张力 / 支线张力 |
| 对撞双方 | 两股相反力量分别是什么 |
| 文本落点 | 它具体落在哪个场景、动作、物件、称呼、空间或结构安排上 |
| 服务哪个核心问题 | 它主要推进哪个核心问题 |
| 写作用途 | 适合作为开头、转折、反证、结尾还是细读入口 |

若某条张力只是概念上成立，落不到文本，就先不要列入主图。第五阶段要的是主次压缩，不是概念铺陈。

## 9. 五异法细读点

第五阶段要从全书里挑出 5 个左右 **五异法** 细读点，尽量覆盖以下几类：

1. 逻辑反常。
2. 情感反常。
3. 字词突兀。
4. 结构断裂或结构转轴。
5. 刻意沉默。

这里的关键，是用五异法找到真正能把问题往下凿的入口，不是为了凑齐五类硬拼材料。若一本书某一类特别少，可以少写，不要造假。

每个细读点至少说明：

| 字段 | 要求 |
|---|---|
| 异常类型 | 属于哪一类五异法 |
| 文本触发点 | 具体词、动作、句式、场景、结构转折或沉默位置 |
| 为什么这里异常 | 它为什么值得停下来，不是一般描写 |
| 它怎样推进核心问题 | 这个细节如何改变、压实或复杂化某个核心问题 |
| 它与张力地图的关系 | 它照亮了哪条主张力或次张力 |

五异法细读点，优先服务于核心问题和张力地图，不要写成互不相干的漂亮摘抄。

## 10. 写作转化

完成核心问题、证据-反证表、张力地图和五异法细读点后，再进入 **写作转化**。第五阶段至少要产出：

1. 2 至 3 个短札题目。
2. 1 至 2 个主题短文方向。
3. 1 个长文提纲。

每个写作方向都必须包含：

| 字段 | 要求 |
|---|---|
| 标题 | 先给可工作的标题，不必追求终稿题目 |
| 核心问题 | 它围绕哪一个核心问题展开 |
| 先抓哪个场景 | 开头先抓哪一幕、哪一个动作、哪一句话 |
| 使用哪些卡片或证据 | 具体调用哪些卡片、场景、张力或细读点 |
| 必须处理的反证 | 若不处理，会让文章立刻变薄的反证是什么 |
| 最容易写坏的地方 | 例如口号化、道德审判、概念堆积、空泛抒情 |
| 如何避免 | 具体改法是什么，例如回到动作链、称呼变化、叙事转折 |

写作转化只给路，不急着写满。若用户没有明确要求成文，不把这些结构自动拼成一篇完成稿。

## 11. 作家写法档案

第五阶段还要形成一个 **作家写法档案**。这里总结的是，这本书里作者怎样写人、写环境、写关系、写转折、写伦理压力，而不是急着下“作者一贯如此”的总定论。

建议至少写这几类：

| 维度 | 要求 |
|---|---|
| 怎样写人 | 作者更常用动作、视角、对话、停顿，还是别的方式把人物立起来 |
| 怎样写环境 | 环境是背景板，还是会反过来挤压人物判断 |
| 怎样写关系 | 关系变化主要通过什么细节显形 |
| 怎样写转折 | 转折靠事件、称呼、空间变化、叙事截断，还是别的结构手法 |
| 怎样写伦理压力 | 压力来自制度、关系、羞耻、体面、欲望，还是多股力量同时挤压 |
| 本书特别突出的写法 | 这是本书此刻最明显的观察 |
| 后续待验证的稳定写法 | 这是值得拿去对照其他作品继续验证的假设，不是定论 |

作家写法档案必须贴文本，不要只写“现实主义很强”“人物很真实”这种空话。

## 12. 跨作品联动

第五阶段的 **跨作品联动**，是为了把当前问题再往前推进，不是为了堆书名显得见识广。

每条联动至少包含：

| 字段 | 要求 |
|---|---|
| 联动作品 | 可以是同一作者，也可以是其他作者 |
| 对照问题 | 两部作品真正拿什么问题对照 |
| 为什么能对照 | 它们在哪个结构、主题、关系或写法层面相通 |
| 对照后推进点 | 这样一比，原核心问题会被推进到哪里 |
| 使用提醒 | 哪些部分可以类比，哪些部分不能偷换 |

如果暂时找不到合适的联动作，不硬凑。宁可留出待补位置，也不要写一串没有推进力的书名。

## 13. 执行顺序

1. 通读 `{正式笔记路径}` 中第四阶段留下的整书结构，确认哪些材料已经足够支撑第五阶段，哪些还只是方向词。
2. 先从 `阅读现场档案` 与 `文章素材索引` 中整理出问题簇，不直接抢跑写标题。
3. 把问题簇压成 3 至 5 个核心问题，并给出问题阶梯与最终推荐表达。
4. 为每个核心问题补齐证据-反证表，尤其要写清 `反证如何修正判断`。
5. 画出张力地图，先列主张力，再压次张力，删去落不到文本的空张力。
6. 选出 5 个左右五异法细读点，把它们挂回核心问题和张力地图。
7. 在此基础上做写作转化，给出短札、主题短文和长文提纲。
8. 最后补作家写法档案与跨作品联动，确认这些模块不是游离附录，而是真的服务当前核心问题。
9. 完成后做自检。自检不过，本阶段不算完成。

## 14. 自检与完成标准

完成前，至少逐项确认：

1. 已明确整本书读完，且第四阶段已经完成。
2. 已先读上下文、先建计划、先过审查，再执行，不是跳过计划门直接开写。
3. 已形成 3 至 5 个核心问题，且核心问题不是主题词或口号。
4. 每个核心问题都有证据-反证表，且写清了 `反证如何修正判断`。
5. 张力地图已经压出主次，不是只列抽象对立词。
6. 五异法细读点能回到具体文本，不是漂亮摘抄拼盘。
7. 写作转化里至少有短札、主题短文方向和长文提纲，不是只给几个空标题。
8. 作家写法档案区分了“本书观察”和“后续待验证假设”，没有把单本书经验写成作家定论。
9. 跨作品联动写清了对照问题和推进点，没有只堆书名。
10. 当前产物仍然是第五阶段结构化结果，不是补卡整理，不是重跑第四阶段，也不是直接完成终稿。

满足以上条件，才算第五阶段完成。
