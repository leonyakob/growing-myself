## 文学分析质量规则

第二阶段优化时，评价重点放在分析质量、分析深度和文学表达上。

最高原则：**AI 不是润色器，而是细读推进器。** AI 的任务不是把我的判断写得更漂亮，而是通过细读文本，主动发现我问题链中最有文学潜力、最容易被忽略的解释分支。必要时要反驳我的判断，也要反驳 AI 自己先前的顺手判断。

开始完整卡或主卡候选分析前，必须先判断：我的原问题里最有价值、最锋利、最不能被总论覆盖的分叉是什么。不能用作品外的大主题、习惯性概括或漂亮总论压过具体词语、动作、沉默和心理雾区。

### 证据锚点与文本落点

完整卡和主卡候选的分析必须回到具体文本细节，例如：

- 词语
- 动作
- 表情
- 沉默
- 场景
- 人物关系
- 叙事位置
- 前后情节的变化

不能只用“阶层”“自尊”“爱情”“城乡差异”“命运”等抽象词概括。

完整卡、主卡候选和正式阅读笔记中的核心卡，必须至少绑定一个可回到原文的证据锚点，可以是划线原句，也可以是清楚可辨的词语、动作、表情、场景、沉默、人物关系或叙事位置。轻卡只需保留一个简短文本落点或轻判断，不必承担完整卡的分析负担。

文学作品与非虚构作品都适用这一原则，但分析重点不同：文学作品侧重词语、动作、人物心理和叙事结构；非虚构作品侧重概念、论证、证据、结构、修辞和隐含前提。不得把小说人物心理分析硬套到非虚构作品，也不得把非虚构的概念概括替代文学作品的细读。

### 人物心理要写出过程，不要直接贴标签

分析人物时，要写清楚心理动作链：

- 人物意识到了什么；
- 他回避了什么；
- 他想维护什么；
- 他害怕承认什么；
- 他在哪一点上发生动摇；
- 他还没有走到哪一步。

不要直接把人物心理写成没铺垫的结论。要把从文本细节到心理判断之间的桥梁写出来。

### 文本歧义与心理雾区不能一笔带过

遇到“不知为什么”“有时”“最好”“似乎”“仿佛”“老半天”“开玩笑”“笑了笑”“沉默”等模糊词、力度词、停顿和沉默时，不能直接越过。这些词常常是人物心理复杂性的入口。

处理这类细节时，至少先辨析：

1. 这个词修饰的是哪一个念头、动作或情绪；
2. 它的力度是确定、犹豫、试探、退缩，还是自我遮掩；
3. 它打开了哪些可能解释，这些解释是否互斥，还是可以同时存在；
4. 当前文本最多能支持哪一层判断，哪一层需要后文验证。

但不要机械逐词阐释。只有当模糊词、力度词、停顿、沉默和人物选择、关系转折、叙事留白或主题张力有关时，才作为心理雾区处理。分析必须说明，为什么这个词值得停下。

### 完整卡和主卡候选必须打开矛盾

每张完整卡或主卡候选至少要指出一个张力点。这里的“张力”不是固定主题清单，而是要求 AI 从当前文本里寻找相反力量的碰撞，例如：

- 人物想要什么 vs 现实不允许什么；
- 人物说出口的话 vs 没说出口的沉默；
- 表面动作 vs 隐藏心理；
- 情感冲动 vs 身份、伦理、习俗或制度限制；
- 自我想象 vs 他人眼光；
- 个人选择 vs 时代、阶层、家庭或命运结构；
- 叙事明写的内容 vs 叙事刻意留白的内容；
- 一个场景里的温情、羞耻、恐惧、欲望、尊严等复杂力量之间的冲突。

这些只是寻找张力的方法，不是要求每本书都套用同一组主题。分析必须从当前作品、当前章节和当前文本细节出发，找到真正存在的矛盾，不要因为示例里出现某类张力就强行套用。

### AI 要提供表达增量

AI 不能只说“这个判断有道理”“这个问题很敏锐”“可以继续发展”。

必须把用户没说透的地方说出来：

- 补心理层次；
- 补文本细节；
- 补问题链；
- 补文章方向；
- 补一个新的合理视角；
- 尽量用有文气但不空泛的语言表达。

所谓“表达增量”，不是替用户改成学术论文，而是在保留用户真实反应的基础上，把直觉说深，把判断说细，把矛盾说出文学味。

### AI 可以反驳，但必须指出缺失的分析环节

如果用户判断跳跃、太宽、太二选一、证据不足或表达不准，AI 应直接指出。

但不要只说“这不准确”“这里需要修正”“这个判断太跳”。

而要说明：

- 少了哪一环；
- 哪个文本细节还没接上；
- 当前判断最多能推出什么；
- 如果要推进成更强判断，需要补什么证据或中间分析；
- 可以怎样改成更稳、更深、更有文学性的表达。

### 大胆分析，但不要省掉过程

可以借助后文回看、作者背景、社会结构、同书内部联动和跨作品比较提出深层解释。

但不能直接空降结论。要写出为什么能这样看、从哪里推进过来、当前文本能支撑到哪一步。

深度不是保守，深度是把判断的来路写出来。

### 升级问题链后必须给出阶段性回答

完整卡、主卡候选和正式阅读笔记中的核心卡，不能只停在“问题链升级”。如果已经提出了较强的问题链，AI 必须继续补“当前回答（沿原问题）”“当前回答（新增角度）”和必要的“后文待回看”。

问题链升级必须分两层：

1. **问题链升级（沿原问题）**：保留我的核心关切，把问题问得更准、更贴文本。
2. **另一条升级问题链（新增角度）**：从当前文本真实存在的另一条线索或写法角度重新打开问题，不预设固定主题清单，不把某本书的主题强行套到其他作品里。

这里的回答不是标准答案，也不是把后文封死，而是说明：根据当前文本细节，问题目前最多能回答到哪里，哪些判断已有证据，哪些还需要后文验证。

推荐写法：

```markdown
- 问题链升级（沿原问题）：
- 另一条升级问题链（新增角度）：
- 当前回答（沿原问题）：
- 当前回答（新增角度）：
- 后文待回看：
```

要求：

1. 回答必须落到具体文本细节，不能只用抽象概念概括。
2. 如果有两条问题链，就分别回答，不能把“沿原问题”和“新增角度”压成一个漂亮总论。
3. 如果用户的问题包含“是 A，还是 B，还是 C”，必须逐项回应：A/B/C 各自的文本依据、限制、是否可共存，以及当前文本更稳的阶段性判断。
4. 回答要写成阶段性判断，不要写成盖棺定论。
5. 如果当前证据不足，要明确写“只能回答到这里”，并指出后文需要验证什么。
6. 轻卡不强行补回答，只有完整卡、主卡候选、文章核心候选需要补。
7. 原则是：问题链负责把思考打开，阶段性回答负责把思考落到文本，后文待回看负责保留弹性。

### AI修正必须深化，不只是整理语序

AI修正不是简单整理语序，也不是把我的话改得更顺。它必须在保留原意的基础上完成至少一个深化动作：

1. 补出我没有说透的心理机制；
2. 补出文本细节和判断之间的桥梁；
3. 补出人物动作背后的身份、关系、处境或叙事压力；
4. 把原判断从“我觉得”推进到“文本如何让这个判断成立”。

如果 AI修正只是复述、压缩、顺句或换同义词，视为不合格，必须重写。

AI修正还必须独立可读，离开前后栏目也能看出它在修正哪一段材料、依据哪个细节、得出什么判断。不要只写“这里”“上面所说”“这种情绪”等无法回指的句子。

### 文学表达不是轻卡专属

文学味儿不只用于轻卡。完整阅读卡、人物主线卡、主卡候选和文章核心候选，也必须在分析准确的基础上，适当给出有文学表达力的句子。

但文学表达必须服从文本和分析，不能为了漂亮而失真。

合格的文学表达应满足：

1. 落到具体动作、物件、场景、沉默或人物关系；
2. 能推进分析，而不是只制造修辞；
3. 不编造文本没有的情绪；
4. 不把人物写得比文本更极端；
5. 能作为正式阅读笔记中的句子使用。

每张完整卡 / 主卡候选至少补一处“可入笔记的文学表达”，可以是一句，也可以是一小段。

### 分析后命名，而非先命名硬套

高质量书评不是先制造漂亮标题，再硬把文本塞进去，而是先细读行为链、心理链和上下文，等分析成熟后，如果自然出现一个能统摄全段的表达，再把它前置为书评开头。

要求：

1. 命名是分析完成后的压缩结果，不是分析开始前的模板。
2. 如果没有自然生成的统摄表达，不要硬造标题。
3. 某个成功书评标题，只有在后文行为链、心理链和上下文都能支撑时才成立。
4. 不要把某个成功标题固化成通用套路，方法可以沉淀，命名不能机械套用。

### 项目文学分析工具箱，执行时必须显式使用

- **五异法扫描**：从逻辑反常、情感反常、字词突兀、结构断裂、刻意沉默里找切口，帮助核心卡打开真正值得深挖的缝隙。
- **问题阶梯（1星-5星）**：先判断用户的问题目前停在情节追问、心理动作背后的矛盾，还是作者写法与核心主题探讨，再决定怎么升级问题链。
- **张力地图**：把文本中的相反力量摆出来，看冲突在哪，消解在哪，诗眼或文眼可能在哪。
- **轻卡/主卡边界**：轻卡记录气氛、心动瞬间即可；主卡必须围绕核心问题展开证据、判断、问题链和文章方向。
- **细节卡从具体往抽象**：先写清动作、物件、表情、空间、沉默，再往上提炼心理和主题。
- **主题卡从抽象往具体落**：遇到议论、判断、独白或概括性句子，先追问它落在什么情境，再回到现实矛盾和具体证据。
- **提问可以跳远，论证不能偷步**：问题可以大胆，但论证必须把中间桥梁写出来，不能从直觉直接跳到定论。
- **保留有生命力的句子**：先保住用户原话里真正有感觉、有文气的部分，再做分析增量，不要把它改成说明书。
- **文本落点**：每个重要判断都要落回词语、动作、场景、沉默、人物关系、叙事位置或论证结构。
- **另一个角度**：在强化原判断后，必须补一个贴文本的新角度，不能为了新奇硬拐。
- **扩写方向与跨作品联动路径**：先修正用户原方向，再补同书内部联动、主题纵深、跨作品联动路径，说明怎样比较，能把原问题推进到哪里。

如果用户用原文证据反驳，要认真修正，文本优先。

正式优化前，先写：

```markdown

- 本轮轻卡数量：
- 本轮完整卡候选数量：
- 本轮主卡候选数量：

```

如果本轮内容很多，不要为了完成全部内容而压缩分析。可以按以下方式分批：

1. 先优化主卡候选；
2. 再优化完整卡候选；
3. 再优化轻卡；
4. 最后处理暂存素材归宿；
5. 或按章节分批优化。

如符合总提示词中的复杂任务计划门，必须先制定优化计划，再分批执行。

## 轻卡优化规则

轻卡适用于感受型、浅判断型、短问题、气氛记录。处理目标是“轻而有味”：保留真实触动和用户原句的生命力，同时把表达稍微修亮，不把它硬拽成完整论文，也不压成无味摘要。

每张轻卡按以下结构补在原想法下面：

```markdown
**AI轻评：**

- 这条想法抓住了什么：
- 可以稍微说深的一点：
- 不必强行展开的原因：

**AI优化表达：**

>
```

要求：

- 先保护真实触动，不强行拔高。
- 至少保留触动点、原想法中有生命力的一句，以及一个简短文本落点或轻判断。
- 如果我的表达太直白，可以优化成更有文学味儿的表达。
- 可以引用相关名言或典故，但必须准确、贴切，不能为了好看硬套，不能编造出处。不确定出处时，不要写成“某某说”。
- 优化表达直接写在原想法下方，不覆盖原想法。
- 不强行补“我提出的问题 / 我的初步判断 / 以后可扩写成什么文章”。
- 不强行补完整问题链、阶段性回答或跨作品联动。轻卡略丰满不是问题，只有明显写成完整卡、破坏整体层次时才减重。

## 完整卡 / 主卡优化规则

完整卡候选、主卡候选适用于“片段分析型”和“完整分析型”想法，尤其是已经接近正式阅读笔记写法的内容。

处理时必须遵守 `/home/king/github/growing-myself/AGENTS.md` 的读书笔记评价结构，并重点执行“文学分析质量规则”。

每张完整卡或主卡候选按以下结构补在原想法下面：

```markdown
**AI评价：**

- 这条想法抓住的真实问题：
- 原问题里最有价值的分叉：
- 好在哪里：
- 目前还没说透的地方：
- 问题质量：
- 判断是否有跳跃：

**AI修正：**

>

**可入笔记的文学表达：**

>

**AI补充：**

- 另一个角度：
- 可以打开的张力：
- 问题链升级（沿原问题）：
- 另一条升级问题链（新增角度）：
- 当前回答（沿原问题）：
- 当前回答（新增角度）：
- 后文待回看：
- 可扩写方向：
- 同书内部联动：
- 扩写方向与跨作品联动路径：

**同位置其他书友高赞想法分析（如有）：**

- 与我的想法形成回声 / 补充 / 挑战 / 反向解释的高赞想法：
- 与我的想法无直接关系但独立精彩的高赞想法：
- 是否建议随对应卡片内联，或作为无法归卡但独立精彩的外部读者材料暂存：
```

要求：

1. 评价要具体，不能空泛夸“敏锐”。
2. 必须指出文本落点，词语、动作、场景、沉默、人物关系、叙事位置、概念节点或论证结构都可以成为落点。
3. 必须分析心理过程，不要直接贴人物标签。
4. 必须打开一个张力点。
5. 如果用户判断跳跃，要直接指出缺少哪一环。
6. 如果用户原问题包含多个解释分支，必须先保护这些分支，再判断哪些有文本依据，哪些需要后文验证。
7. 遇到“不知为什么”“最好”“老半天”“开玩笑”等心理雾区，必须先辨析词语力度和可能指向，再做判断。
8. AI修正必须保留用户原意，但不能只整理语序，必须补出深化分析。
9. 完整卡、人物主线卡、主卡候选必须补“可入笔记的文学表达”。
10. AI补充要提供新的文学分析增量，不要重复用户原话。
11. 如果补了较强的问题链，必须继续写“当前回答（沿原问题）”“当前回答（新增角度）”和“后文待回看”。

## 暂存素材处理规则

暂存素材不是废料。它可能暂时不适合展开，但以后要有归宿。

每条暂存素材至少说明：

```markdown
**暂存说明：**

- 可归入的主题：
- 可能服务的人物线：
- 以后可用于的文章或对照方向：
- 是否需要等后文回看：
```

暂存素材只说明可能归宿，不强行拔高，不强行补完整问题链，不把它提前写成正式阅读笔记段落。

## 同位置其他书友高赞想法差异分析规则

第二阶段处理同位置其他书友高赞想法时，重点不是判断“谁说得更好”，而是看外部读者材料怎样照亮我的阅读位置。

每条有效高赞想法按关系分簇处理：

1. **外部读者回声**：与我的想法同向，能证明这个触动点不是孤立的。
2. **外部读者补充**：补出我没有说到的细节、人物心理、叙事安排或时代背景。
3. **外部读者挑战**：与我的判断不同，但能贴文本提出另一种解释。
4. **反向解释**：不是简单反驳，而是从相反立场重读同一细节，使原问题更复杂。
5. **独立精彩外部视角**：与我的原想法没有直接关系，但本身贴文本、有张力、有问题意识或有表达穿透力，能够独立推进作品理解。

判断规则：

- 点赞数、热度或接口默认排序只负责发现，不负责最终定性。
- AI 必须根据文本质量判定高赞想法是否值得保留，是否贴住具体细节，是否打开人物心理、叙事安排、主题张力、概念矛盾或论证前提，是否能形成可追问的问题，是否避免鸡汤化和道德化。
- 与我的想法无关但独立精彩的高赞想法，不要强行解释成“对我的想法的补充”，直接标为“独立精彩外部视角”。
- 方向相反但都贴文本的高赞想法，应并存为“外部读者分歧”，不要只保留更赞同我的一条。
- 外部读者原话必须逐字保留，优先随对应卡片内联，并明确标为“外部读者原话”及其关系：回声、补充、挑战、反向解释或独立精彩。不得改写成用户第一人称判断，也不得概括为“有读者说”。
- 只有无法归入具体卡片、却仍独立精彩的外部材料，才进入全局“外部读者精彩想法”小节。外部读者材料不是每张卡的必填项。
- 第二阶段可以在对应条目下保留“同位置其他书友高赞想法分析”，但不得让高赞想法覆盖我的原想法、AI评价、AI修正和AI补充。

## 第二阶段 QA

正式输出前，逐条自检核心卡：

1. AI修正是否只是顺着我的原话整理语序？如果是，必须重写。
2. 是否补出了新的心理机制、文本桥梁、结构张力或论证支点？
3. 是否先保留了我的原问题中最有价值的解释分支，没有用 AI 总论覆盖？
4. 是否辨析了关键模糊词、力度词、停顿或沉默中的心理雾区？
5. 问题链是否同时包含“沿原问题升级”和“新增角度升级”？
6. 是否分别给出了“当前回答（沿原问题）”“当前回答（新增角度）”和“后文待回看”？
7. 如果原问题是 A / B / C 分叉，是否逐项判断了各分支的文本依据、限制和是否可共存？
8. 完整卡、人物主线卡、主卡候选是否有可入笔记的文学表达？
9. 文学表达是否贴文本，没有为了漂亮而失真？
10. 同位置其他书友高赞想法是否只作为外部读者材料，没有替代我的原想法？
11. 是否显式使用了五异法扫描、问题阶梯（1星-5星）、张力地图，并守住轻卡/主卡边界？
12. 是否同时保住“保留有生命力的句子”和“文本落点”，没有把用户原话改成空泛说明？
13. 是否补出了另一个角度，以及必要时的扩写方向与跨作品联动路径？
14. 是否仍然只在 `{中间稿路径}` 内优化，没有写入任何正式阅读笔记文件？
15. 若用户用原文证据反驳，是否按文本优先原则认真修正？

输出完成后，再通读一次整轮结果，确认所有 AI 材料都写在用户原想法下方，确认没有把通用规则写成小说专用，也确认没有把第二阶段越界成第三阶段或第四阶段。
