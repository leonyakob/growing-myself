- 后来出现的证据：`第4轮 / 一、文章核心候选（15） / 1. 借来的权力光：高玉德为什么非要带弟弟去吃这顿饭`、`第4轮 / 二、完整阅读卡（67） / 3. 马占胜搂肩承诺工作：关系链如何让办事速度突然激活`、`第4轮 / 二、完整阅读卡（67） / 58. 走后门被查实和叔父退回去：制度清算节点` 把关系链写得更完整。
- 全书后的修正：高加林当然恨压到自己头上的特权，但他也一再被“如果这份特权能给我用呢”这个问题诱惑，所以全书里更刺人的不是他单纯受害，而是他也痛恨自己没有特权。
- 误读的价值：早期只盯制度不盯欲望，反而帮我看清后来最该追的是“受害者怎样想复制加害者的位置”。
- 文章索引用法：可同时挂到“人物线”和“尊严主题”，写高加林的心气如何被关系权力不断拧歪。

#### 劳动与身份：从“高加林不愿掏炭”推进到他能承受新闻现场之苦、真正差别在身份意义
- 当时的读法：先把 `第4轮 / 二、完整阅读卡（67） / 5. “加林恐怕不愿去掏炭”：体面劳动与身份回落` 读成高加林吃不了苦。
- 后来出现的证据：`第4轮 / 二、完整阅读卡（67） / 11. 喝路边水坑、脚流血：高加林肯为被认可的前途吃苦`、`第4轮 / 二、完整阅读卡（67） / 12. 第一个到达灾区：个体积极与系统迟缓`、`第4轮 / 二、完整阅读卡（67） / 15. 会议上毫不拘束提问：高加林的职业适配` 把另一种吃苦写得很足。
- 全书后的修正：他不是不能吃苦，而是不肯为一个让自己觉得掉价的身份吃苦；一旦苦和“我真正像个新闻记者了”连上，痛感就被重新编码了。
- 误读的价值：先把他看扁，后来才更能看清“怕苦”和“怕掉回某种位置”不是一回事。
- 文章索引用法：主挂“尊严主题”，也能接到“知识分子困境”，用来写意义怎样改写劳动感受。

#### 职业热爱：从“热爱让苦甘之如饴”收稳为记者身份重新编码痛感，并保留田晓霞联想但不混同两人
- 当时的读法：看见灾区奔跑、喝脏水、脚流血后的愉快，第一反应是“做热爱的事情再苦再累都是甘之如饴”，并自然想起田晓霞。
- 后来出现的证据：`第3轮 / 三、完整阅读卡 / 2. 现代文明的风：高加林为什么没有先成为传播者`、`第4轮 / 二、完整阅读卡（67） / 11. 喝路边水坑、脚流血：高加林肯为被认可的前途吃苦`、`第4轮 / 二、完整阅读卡（67） / 16. 第一篇报道被播出：职业承认和景老师一字未改` 说明这里还有强烈的自我证明。
- 全书后的修正：这当然有职业热爱，但更准确的是，记者身份让他终于找到了肯认的苦，所以不只是“热爱”，还是职业身份把痛感重新编了码。
- 误读的价值：先想到田晓霞并不丢人，它提醒我高加林身上也有被公共现场点燃的一面，只是他的亮更容易回到“我终于有用”这件事上。
- 文章索引用法：主挂“知识分子困境”，副挂“写法线索”，写路遥怎样用身体苦把职业热情和身份确认压在一起。

#### 县城空间：从“县城变小等于忘本”修正为身份落差投向空间的心理投影，不把后来的失败倒灌进当时场景
- 当时的读法：先看到“县城变小、陌生”，就顺手骂成“进县而忘村，去市就嫌县”，甚至直接判成忘本。
- 后来出现的证据：`第1轮 / 一、文章核心候选 / 2. 读书之后回不了土地：身份坠落与县城启蒙`、`第4轮 / 一、文章核心候选（15） / 11. 县城变小、陌生：见过更大世界后的空间失衡`、`第4轮 / 二、完整阅读卡（67） / 10. “我再也不能离开你了”：县城像恋人，也像审判者` 把空间参照系的变化写得更清楚。
- 全书后的修正：县城没有自己变小，是高加林见过更大的世界以后，把身份落差和欲望升级投到了空间上；那一刻还不能直接倒灌后来的失败感进去。
- 误读的价值：早一点喊“忘本”，反而逼我后来把时间顺序和心理顺序分开，不让结果偷跑回前面的场景。
- 文章索引用法：主挂“城乡主题”，可写路遥怎样把空间经验写成人物的欲望刻度。

#### 爱情与位置：保留“农民时爱巧珍、干部时爱黄亚萍”的原始锋利表达，同时补足“位置影响哪一种爱被承认”的证据桥
- 当时的读法：最先抓住的就是那句刺人的判断，农民时爱巧珍，干部时爱黄亚萍，觉得这已经够说明问题。
- 后来出现的证据：`第2轮 / 一、文章核心候选 / 4. 不公开恋爱，却宣称巧珍已有对象：高加林爱情里的权利和责任`、`第3轮 / 一、文章核心候选 / 1. 同车不同心：巧珍的爱与高加林的报复`、`第4轮 / 一、文章核心候选（15） / 4. 黄亚萍的爱情价值地图：爱一个人，也是爱一条去远方的路`、`第4轮 / 一、文章核心候选（15） / 13. “我真正爱的人实际上是另外一个”：更爱巧珍里的真情、偏见和自我辩护` 把这句话背后的位置逻辑补厚了。
- 全书后的修正：这句锋利话要保留，但不能只停在骂他“变心快”。更稳的是，看见高加林总把爱和位置绑在一起，于是哪一种爱能公开、能想象未来、能被他说成“真正”，就不断随位置移动。
- 误读的价值：原来的狠话保住了问题尖度，后来的补证才让我没有把人物压扁成一句口号。
- 文章索引用法：主挂“爱情线”，也能接到“人物线”，写爱怎样被前途、体面和身份解释权改写。

#### 黄亚萍物质付出：从“这不是牺牲”修正为“有真实成本，但不等同于巧珍在匮乏中的托举”
- 当时的读法：先对“牺牲”这两个字起反感，觉得黄亚萍物质上本来就宽裕，这不能算牺牲。
- 后来出现的证据：`第4轮 / 一、文章核心候选（15） / 4. 黄亚萍的爱情价值地图：爱一个人，也是爱一条去远方的路`、`第4轮 / 二、完整阅读卡（67） / 45. 进口削苹果刀丢了：精致物件与利己试探`、`第4轮 / 三、人物主线卡（只承载人物弧线补证） / 3. 黄亚萍人物弧线：市民气息、自我中心、冷感语言和台阶`，再加上那一整段衣服、皮鞋、手表、糖果，说明她并非零成本。
- 全书后的修正：更稳的说法不是“这不是牺牲”，而是她确有真实付出，也确有情意，但这种成本不等同于巧珍那种从匮乏里往外掏、并把自己一并押上的托举。
- 误读的价值：先把“牺牲”顶回去，帮我避开了把富裕赠与浪漫化的陷阱；后来补一层真实成本，才不至于把黄亚萍写得太轻薄。
- 文章索引用法：主挂“爱情线”，也可接“人物线”，写两种付出背后的阶层条件和关系代价。
