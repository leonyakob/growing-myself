## 2. 选择标准，不按 `星级` 机械排座次

本轮不是挑“最顺手写”的三题，也不是把 Todo 3 里标了 5 星的题简单排前三。最终三题同时满足四个条件：

1. **问题深度够 5 星**，能从情节或人物判断进入心理机制、结构代价、写法或主题根部。
2. **A级原卡够厚**，每题至少能稳定回到 3 个以上原卡锚点，不靠 Stage 4 总括句顶着走。
3. **反证面还活着**，问题不能一问就只剩道德裁决，必须能容纳复杂性和回头修正。
4. **彼此不重题**，三题不能共享同一中央判断，也不能共享过多主锚点。每题都要能单独支撑 Todo 7 到 Todo 9 的后续展开。

### 5.1 机械边界结论

- `FQ-01` 与 `FQ-02` 的主锚点没有重合到两处以上。前者主打空间 / 职业 / 远方链，后者主打履历表 / 刷牙 / 掏炭 / 庄稼地里链。
- `FQ-01` 与 `FQ-03` 的主锚点也没有重合到两处以上。前者问安身位置，后者问关系责任和村庄代价。
- `FQ-02` 与 `FQ-03` 都碰爱情，但一个问内部机制，一个问公共后果，主锚点分别落在 `567/2446/3154` 和 `700/1248/1908/2029`，不是同一题拆排比句。
