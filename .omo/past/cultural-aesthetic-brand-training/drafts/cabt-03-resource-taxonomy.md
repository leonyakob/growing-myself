# 资源系统与选择逻辑片段

本片段只定义资源系统的分类、层级和选择逻辑，不列最终推荐书单，也不把名气当成选择理由。后续具体资源必须服从这里的判断：资源不是为了收集事实而存在，而是为了训练感知、判断、语言、具身观察和个人精华提取。

资源进入训练方案前，先问五个问题：它能让人看见什么？它能纠正什么浅层理解？它能让用户说出哪一种“为什么是我这样理解”？它能否进入 10-20 分钟中长视频的论点、场景或语言？它是否服务前 90 天的重点，文学文化与器物空间？

# 一、五层资源系统

## 1. root texts / 根文本 / 原典

根文本是文化训练的第一层，包括经典文本、文学作品、原典语句、历史语境里的核心观念。它们不只是提供知识背景，而是训练用户直接接触语言、人物、价值冲突和审美秩序的能力。

这一层的选择标准是：语言有密度，人物或观念有可反复解释的空间，能让用户从“我知道它”走向“我被它怎样击中”。根文本优先进入前 90 天，因为文化根没有长稳，后面的现代转译和审美表达都会变成空泛包装。

## 2. interpretive bridges / 解释桥梁

解释桥梁是第二层，包括注释、导读、学者讲解、访谈、讲座、评论文章和可靠的博客/专栏。它们帮助用户理解语境、概念和争议，但不能替代根文本。

这一层训练的是判断力：知道什么时候需要帮助，什么时候要回到自己的阅读。好的解释桥梁会打开问题，不会把文化压成标准答案。

## 3. embodied culture / 具身文化

具身文化是第三层，包括器物、空间、茶、瓷、花、园林、服饰、饮食、仪式、材料、工艺和日常生活现场。它让文化从文本进入身体经验，让用户观察触感、尺度、秩序、留白、使用方式和人与物的关系。

这一层尤其服务前 90 天的器物空间训练。它防止东方美学被讲成装饰符号，也让中长视频有可落地的场景、画面和生活记忆。

## 4. modern translation resources / 现代转译资源

现代转译资源是第四层，包括当代写作、现代中文表达、文化评论、城市生活观察、现代女性经验、心理与关系议题、设计与生活方式表达。它们帮助用户把古典经验转成当代人能进入的情绪、关系、时间焦虑、秩序感、品味和日常仪式。

这一层的任务不是把文化讲得更流行，而是让用户找到古今之间的真实连接。选择时要避开空泛鸡汤和流量化表达。

## 5. output practice references / 输出练习参照

输出练习参照是第五层，包括写作工艺、长文结构、纪录片叙事、视频讲述、访谈、讲座表达、博客/专栏写法和 10-20 分钟中长视频拆解。它们服务最后的表达训练，不应该反过来决定文化根。

这一层训练语言、结构、节奏和转场，让个人精华能被组织成观众愿意听完的内容。它必须晚于文化阅读和个人理解，不抢根部位置。

# 二、资源选择总原则

1. 先根部，后表达。先文学文化与器物空间，再进入审美表达、AI 协作和品牌/频道。
2. 先训练感知，再补充知识。资源如果只增加谈资，却不能训练观察、判断或语言，就不优先。
3. 先提取个人精华，再做现代转译。用户要能说清“我为什么被它击中”，而不是只复述权威解释。
4. 先多类型输入，再统一输出。书、视频、纪录片、讲座、博客/专栏、博物馆/展览/图录、访谈和田野观察都可以进入训练，但每一种都要有明确任务。
5. 高端/奢侈意义建构只能放在文化根和解释桥梁之后。它用来理解意义如何被组织，不用来替代文化素养。

# 三、资源类别与训练目标

## 1. 中国经典与古典文学

这类资源包括诗、词、文、小说、思想原典和能承载文化气质的古典文学。此处不列具体推荐，后续只选择能反复进入解释的文本，而不是因为它们有名。

- 训练什么感知：训练语言密度、人物命运、道德张力、时间感、自然观和礼法秩序的感知。
- 提取什么精华：提取被某句话、某个人物、某种处境击中的私人经验，提取“古典经验怎样照见今天的关系、时间、女性自我和生活秩序”。
- 选择逻辑：优先选能让用户反复追问的文本。若只能形成知识点或名句摘抄，就暂不优先。

## 2. 现代中国文学

这类资源承担古典与当代之间的过渡。它让用户看到中文如何写现代人的孤独、家族、城市、女性、身体、阶层、记忆和自我变化。

- 训练什么感知：训练现代中文里的情绪层次、人物复杂性、时代质感、生活细节和叙述节奏感知。
- 提取什么精华：提取现代人处境里的文化余温，提取私人记忆、关系伤口、城市经验和自我选择如何成为可讲述的主题。
- 选择逻辑：优先选能把个人经验带回来的作品。只提供时代背景或文学史标签的资源不优先。

## 3. 美学与哲学

这类资源帮助用户形成审美判断，而不是堆叠“高级”“松弛”“东方感”等空词。它可以来自中国美学、日常美学、生活哲学、艺术哲学和价值判断训练。

- 训练什么感知：训练秩序、留白、节制、情绪收束、价值分寸和美感边界的感知。
- 提取什么精华：提取“我认为什么是好的生活”“我为什么接受或拒绝某种美”的判断精华，提取可稳定审美表达的原则。
- 选择逻辑：优先选能帮助用户说清判断标准的资源。若只制造术语感或学术距离，就不适合作为早期核心。

## 4. 器物与物质文化

这类资源包括茶、瓷、香、花、家具、服饰、文房、园林材料、工艺、器形、纹样和日常使用方式。重点不是把它们摆进画面，而是理解人怎样通过物建立秩序、亲密、克制和时间感。

- 训练什么感知：训练触感、尺度、材料、工艺、使用痕迹、空间位置和人与器物关系的感知。
- 提取什么精华：提取器物如何触发生活记忆、审美判断、身体经验和价值选择，提取“这个物为什么不只是道具”。
- 选择逻辑：优先选能观察、能触摸、能比较、能进入田野观察的资源。只提供漂亮图片或符号标签的不优先。

## 5. 当代文化批评

这类资源包括文化评论、社会观察、女性议题、消费文化、城市生活、媒介批评、博客/专栏和长访谈。它让用户避免把传统文化讲成孤立古物，也避免把现代转译讲成泛泛安慰。

- 训练什么感知：训练时代问题、消费语境、身份焦虑、媒介表达、现代关系和大众情绪的感知。
- 提取什么精华：提取“这个文化材料今天为什么还刺痛人”的问题精华，提取现代观众真正关心的关系、时间、自我和秩序难题。
- 选择逻辑：优先选能提出问题、帮助辨别当代处境的资源。只追热点、立场过满或语言粗糙的内容不优先。

## 6. 高端/奢侈意义建构

这类资源不是训练的根部，只放在文化根、解释桥梁和具身文化之后。它关注高端品牌、奢侈品、工艺叙事、空间仪式、客户想象和价值语言如何组织意义。

- 训练什么感知：训练稀缺性、仪式感、工艺叙事、价值语言、距离感和长期信任的感知。
- 提取什么精华：提取“意义如何被慢慢建立”的结构精华，提取哪些表达能让文化厚度进入高端语境而不变成炫耀。
- 选择逻辑：只选能解释意义建构的资源，不选单纯展示价格、身份、包装或视觉风格的内容。它不能超过文化根部的权重。

## 7. 写作与输出工艺

这类资源包括散文、长文结构、脚本结构、叙事节奏、修辞、讲述训练、纪录片脚本、访谈稿和中长视频拆解。它帮助用户把个人精华写成能被听懂、能被看完、能被记住的内容。

- 训练什么感知：训练开场、论点推进、转折、段落节奏、例子选择、隐喻、情绪落点和结尾余味的感知。
- 提取什么精华：提取表达方法上的精华，如一条观点怎样展开，一段材料怎样变成场景，一次私人感受怎样落到观众经验。
- 选择逻辑：优先选能被拆解和模仿结构的资源。只提供模板话术、短视频钩子或流量套路的不优先。

## 8. 视频、纪录片、讲座、博客/专栏

这类非书资源补足文本之外的声音、画面、节奏和现场感。视频和纪录片训练观看，讲座训练概念铺陈，博客/专栏训练个人化表达与连续观点。

- 训练什么感知：训练镜头里的空间秩序、讲述节奏、口语温度、材料组织、观点连续性和观众进入路径的感知。
- 提取什么精华：提取“怎样把文化讲得可进入”的表达精华，提取能服务 10-20 分钟中长视频的结构、转场和语气。
- 选择逻辑：优先选有清晰问题意识、有现场细节、有审美克制的内容。只漂亮、只煽情、只快节奏的信息流不优先。

## 9. 博物馆、展览与图录

这类资源把文本、器物、空间和历史语境放在同一现场。博物馆、展览、图录可以训练用户看一件物如何被命名、陈列、解释和连接时代。

- 训练什么感知：训练陈列逻辑、空间动线、器物比较、说明文字、材料细节和历史尺度的感知。
- 提取什么精华：提取“物与人怎样互相说明”的观察精华，提取可进入视频场景、器物叙事和审美判断的具体细节。
- 选择逻辑：优先选能让用户现场观察或反复查阅图录的资源。只打卡式展览、只拍照不观察的内容不优先。

## 10. 田野观察

田野观察包括茶席、器物店、展厅、园林、旧街区、书店、手工艺现场、日常餐桌、空间使用和与人的访谈。它不是旅游或采风，而是把文化训练放回真实生活。

- 训练什么感知：训练身体在空间中的感受、人与物的使用关系、光线、声音、气味、等待、礼节和日常秩序的感知。
- 提取什么精华：提取生活现场里的具身精华，提取一件物、一个空间、一次对话或一次停留怎样触发私人理解和现代转译。
- 选择逻辑：优先选可重复观察、可记录细节、能产生个人判断的现场。只追求稀奇或审美打卡的不优先。

# 四、前 90 天资源优先级

前 90 天只抓两条主线：文学文化与器物空间。文学文化负责文化根、语言和人物处境；器物空间负责身体经验、生活秩序和审美落地。解释桥梁可以辅助，但不能压过根文本；现代转译可以练习，但不急着追求成熟品牌表达；高端/奢侈意义建构只作为轻量旁观，不进入主线。

这个排序的目的，是让用户先长出自己的文化感受和判断。等个人精华库、器物空间观察和现代转译能力稳定后，AI 风格学习、中长视频结构和品牌/频道语言才有可继承的根。

# 五、排除标准

以下资源暂不优先：只因有名而被列入的资源，只提供事实背景却不训练感知的资源，只能产出摘抄和知识点的资源，把茶、瓷、空间当成装饰符号的资源，过早引向品牌包装、商业化、短视频流量或身份炫耀的资源。

后续具体推荐必须说明：训练什么感知，提取什么精华，如何使用，能产出什么内容物。说不清这四项的资源，不进入核心推荐。
