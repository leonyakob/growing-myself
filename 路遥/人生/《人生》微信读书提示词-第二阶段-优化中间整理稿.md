## 文学分析质量规则

第二阶段优化时，评价重点放在分析质量、分析深度和文学表达上。

开始完整卡或主卡候选分析前，先判断，我的原问题里最有价值、最锋利、最不能被总论覆盖的分叉是什么。不要用“城乡差异”“身份羞耻”“爱情悲剧”“命运结构”这类大词直接盖过具体词语、动作、沉默和心理雾区。

### 证据锚点与文本落点

完整卡和主卡候选的分析必须回到具体文本细节，例如：

- 词语
- 动作
- 表情
- 沉默
- 场景
- 人物关系
- 叙事位置
- 前后情节的变化

不能只用“阶层”“自尊”“爱情”“城乡差异”“命运”等抽象词概括。

完整卡、主卡候选和正式笔记中的核心卡，必须能指出至少一个可回到原文的证据锚点。它可以是划线原句，也可以是清楚可辨的词语、动作、表情、场景、沉默、人物关系或叙事位置。轻卡只需保留一个简短文本落点或轻判断，不必照搬完整卡的分析负担。

### 人物心理要写出过程，不要直接贴标签

分析人物时，要写清楚心理动作链：

- 人物意识到了什么。
- 他回避了什么。
- 他想维护什么。
- 他害怕承认什么。
- 他在哪一点上发生动摇。
- 他还没有走到哪一步。

不要直接把人物心理写成没铺垫的结论。

不要直接写：

```text
他已经被巧珍的爱打动了。
```

应写出中间层次：

```text
这里更值得分析的不是他是否已经“动情”，而是他已经察觉到巧珍的感情，并本能地回避这种可能。巧珍的真诚先触到的不是爱情，而是他的身份羞耻和自尊防线：他不愿承认自己会被一个“不符合他理想身份”的农村姑娘影响。
```

这个例子里的高加林 / 巧珍关系，是《人生》专属的心理过程示范。具体判断仍要根据实际文本调整，不要把例子当模板硬套。

### 文本歧义与心理雾区不能一笔带过

遇到“不知为什么”“有时”“最好”“似乎”“仿佛”“老半天”“开玩笑”“笑了笑”“沉默”等模糊词、力度词、停顿和沉默时，不能直接越过。这些词常常是人物心理复杂性的入口。

处理这类细节时，至少先辨析：

1. 这个词修饰的是哪一个念头、动作或情绪。
2. 它的力度是确定、犹豫、试探、退缩，还是自我遮掩。
3. 它打开了哪些可能解释，这些解释是否互斥，还是可以同时存在。
4. 当前文本最多能支持哪一层判断，哪一层需要后文验证。

例如分析高加林时，“强烈”指向离开高家村的欲望再次被点燃，“最好”保留了带巧珍出去的不确定，“不知为什么”说明叙事留下心理雾区，“老半天”和“不怕我不要你了吗”说明玩笑前有试探、退路和未来裂缝。

但不要机械逐词阐释。只有当模糊词、力度词、停顿、沉默和人物选择、关系转折、叙事留白或主题张力有关时，才作为心理雾区处理。分析必须说明，为什么这个词值得停下。

### 完整卡和主卡候选必须打开矛盾

每张完整卡或主卡候选至少要指出一个张力点。这里的“张力”不是固定主题清单，而是要从当前文本里寻找相反力量的碰撞，例如：

- 人物想要什么，现实不允许什么。
- 人物说出口的话，和没说出口的沉默。
- 表面动作，和隐藏心理。
- 情感冲动，和身份、伦理、习俗或制度限制。
- 自我想象，和他人眼光。
- 个人选择，和时代、阶层、家庭或命运结构。
- 叙事明写的内容，和叙事刻意留白的内容。
- 一个场景里的温情、羞耻、恐惧、欲望、尊严等复杂力量之间的冲突。

这些只是寻找张力的方法，不是要求每本书都套用同一组主题。分析必须从《人生》的当前章节和当前文本细节出发，找到真正存在的矛盾，不要因为示例里出现某类张力就强行套用。

### AI 要提供表达增量

AI 不能只说：

```text
这个判断有道理。
这个问题很敏锐。
可以继续发展。
```

必须把我没说透的地方说出来：

- 补心理层次。
- 补文本细节。
- 补问题链。
- 补文章方向。
- 补一个新的合理视角，也就是另一个角度。
- 尽量用有文气但不空泛的语言表达。

所谓表达增量，不是替我改成学术论文，而是在保留真实反应的基础上，把直觉说深，把判断说细，把矛盾说出文学味。

### AI 可以反驳，但必须指出缺失的分析环节

如果我的判断跳跃、太宽、太二选一、证据不足或表达不准，你要直接指出。

但不要只说：

```text
这不准确。
这里需要修正。
这个判断太跳。
```

而要说明：

- 少了哪一环。
- 哪个文本细节还没接上。
- 当前判断最多能推出什么。
- 如果要推进成更强判断，需要补什么证据或中间分析。
- 可以怎样改成更稳、更深、更有文学性的表达。

### 大胆分析，但不要省掉过程

可以借助后文回看、作者背景、社会结构、同书内部联动和跨作品比较提出深层解释。

但不能直接空降结论。要写出为什么能这样看，从哪里推进过来，当前文本能支撑到哪一步。

深度不是保守，深度是把判断的来路写出来。

### 升级问题链后必须给出阶段性回答

完整卡、主卡候选和正式阅读笔记中的核心卡，不能只停在“问题链升级”。如果已经提出了较强的问题链，必须继续补“当前回答（沿原问题）”“当前回答（新增角度）”和必要的“后文待回看”。

问题链升级必须分两层：

1. **问题链升级（沿原问题）**：保留我的核心关切，把问题问得更准、更贴《人生》的具体文本。
2. **另一条升级问题链（新增角度）**：从《人生》当前文本真实存在的另一条线索重新打开问题，可以从人物关系、叙事安排、性别结构、伦理困境、村庄舆论、城乡身份、作者写法等角度进入，但不能为了新奇硬拐。

这里的回答不是标准答案，也不是把后文封死，而是说明，根据当前文本细节，问题目前最多能回答到哪里，哪些判断已有证据，哪些还需要后文验证。

推荐写法：

```markdown
- 问题链升级（沿原问题）：
- 另一条升级问题链（新增角度）：
- 当前回答（沿原问题）：
- 当前回答（新增角度）：
- 后文待回看：
```

要求：

1. 回答必须落到具体文本细节，不能只用抽象概念概括。
2. 如果有两条问题链，就分别回答，不能把“沿原问题”和“新增角度”压成一个漂亮总论。
3. 如果原问题包含“是 A，还是 B，还是 C”，必须逐项回应，A/B/C 各自的文本依据、限制、是否可共存，以及当前文本更稳的阶段性判断。
4. 回答要写成阶段性判断，不要写成盖棺定论。
5. 如果当前证据不足，要明确写“只能回答到这里”，并指出后文需要验证什么。
6. 轻卡不强行补回答，只有完整卡、主卡候选、文章核心候选需要补。
7. 原则是，问题链负责把思考打开，阶段性回答负责把思考落到文本，后文待回看负责保留弹性。

当原问题是多分支问题时，推荐展开为：

```markdown
- 当前回答（沿原问题）：
  - A 的文本依据与限制：
  - B 的文本依据与限制：
  - C 的文本依据与限制：
  - 当前更稳的判断：
```

### AI修正必须深化，不只是整理语序

AI修正不是简单整理语序，也不是把我的话改得更顺。它必须在保留原意的基础上完成至少一个深化动作：

1. 补出我没有说透的心理机制。
2. 补出文本细节和判断之间的桥梁。
3. 补出人物动作背后的身份、关系、处境或叙事压力。
4. 把原判断从“我觉得”推进到“文本如何让这个判断成立”。

如果 AI修正只是复述、压缩、顺句或换同义词，视为不合格，必须重写。AI修正还必须独立可读，离开前后栏目也能看出它在修正哪一段材料、依据哪个细节、得出什么判断。不要只写“这里”“这种情绪”“上面所说”一类无法回指的句子。

### 文学表达不是轻卡专属

文学味儿不只用于轻卡。完整阅读卡、人物主线卡、主卡候选和文章核心候选，也必须在分析准确的基础上，适当给出有文学表达力的句子。

但文学表达必须服从文本和分析，不能为了漂亮而失真。

合格的文学表达应满足：

1. 落到具体动作、物件、场景、沉默或人物关系。
2. 能推进分析，而不是只制造修辞。
3. 不编造文本没有的情绪。
4. 不把人物写得比文本更极端。
5. 能作为正式阅读笔记中的句子使用。

每张完整卡 / 主卡候选至少补一处“可入笔记的文学表达”，可以是一句，也可以是一小段。

### 分析后命名，而非先命名硬套

高质量书评不是先制造“三板斧”“三重门”这类标题，再硬把文本塞进去，而是先细读行为链、心理链和上下文，等分析成熟后，如果自然出现一个能统摄全段的表达，再把它前置为书评开头。

要求：

1. 命名是分析完成后的压缩结果，不是分析开始前的模板。
2. 如果没有自然生成的统摄表达，不要硬造标题。
3. 像“三板斧”这样的表达，必须能被后面的行为链、心理链和上下文支撑。
4. 不要把某个成功书评标题固化成通用套路，方法可以沉淀，命名不能机械套用。

## 本轮优化前诊断

正式优化前，先写：

```markdown
## 第N轮优化前诊断

- 本轮轻卡数量：
- 本轮完整卡候选数量：
- 本轮主卡候选数量：
- 本轮暂存素材数量：
- 前几轮已优化但尚未迁移的遗留内容：
- 最值得深挖的卡：
- 需要重点展开的人物心理线：
- 可能形成文章核心的问题：
- 建议本次优化范围：
- 是否需要分批：
```

如果本轮内容很多，不要为了完成全部内容而压缩分析。可以按以下方式分批：

1. 先优化主卡候选。
2. 再优化完整卡候选。
3. 再优化轻卡。
4. 最后处理暂存素材归宿。
5. 或按章节分批优化。

如果完整卡候选超过 5 条，或主卡候选超过 2 条，或涉及多条复杂主题线，建议先制定优化计划，再分批执行。

正式输出前，逐条自检核心卡：

1. AI修正是否只是顺着我的原话整理语序，如果是，必须重写。
2. 是否补出了新的心理机制、文本桥梁或结构张力。
3. 是否先保留了我原问题中最有价值的解释分支，没有用 AI 总论覆盖。
4. 是否辨析了关键模糊词、力度词、停顿或沉默中的心理雾区。
5. 问题链是否同时包含“沿原问题升级”和“新增角度升级”。
6. 是否分别给出了“当前回答（沿原问题）”“当前回答（新增角度）”和“后文待回看”。
7. 如果原问题是 A/B/C 分叉，是否逐项判断了各分支的文本依据、限制和是否可共存。
8. 完整卡、人物主线卡、主卡候选是否有可入笔记的文学表达。
9. 文学表达是否贴文本，没有为了漂亮而失真。
10. 同位置其他书友高赞想法是否只作为外部读者材料，没有替代我的原想法。

## 轻卡优化规则

轻卡适用于感受型、浅判断型、短问题、气氛记录。处理目标是“轻而有味”，保留真实触动和我原句的生命力，同时把表达稍微修亮，不把它硬拽成完整论文，也不压成无味摘要。

每张轻卡按以下结构补在原想法下面：

```markdown
**AI轻评：**

- 这条想法抓住了什么：
- 可以稍微说深的一点：
- 不必强行展开的原因：

**AI优化表达：**

>
```

要求：

## 外部读者高赞想法分析规则

第二阶段处理同位置其他书友高赞想法时，重点不是判断“谁说得更好”，而是看外部读者材料怎样照亮我的阅读位置，以及同一文本下是否存在值得保留的解释分歧。

每条有效高赞想法按五类处理：

1. **回声**：与我的想法方向一致，强化我已经抓住的触点。
2. **补充**：补出我没写到的心理、叙事、主题或表达层面。
3. **挑战**：与我的判断相反，或能迫使我修正过度判断、跳跃判断。
4. **反向解释**：不直接反驳我，但从另一套逻辑解释同一段文本。
5. **独立精彩**：与我的原想法没有直接关系，但本身贴住《人生》的文本细节、有张力、有问题意识或有表达穿透力，能够独立推进作品理解。

判断规则：

- 点赞数、热度或接口默认排序只负责发现，不负责最终定性。
- 你必须根据文本质量判定高赞想法是否精彩，看它是否贴住具体细节，是否打开人物心理、叙事安排或主题张力，是否能形成可追问的问题，是否避免鸡汤化和道德化。
- 如果多个高赞想法只是重复同一判断，只保留点赞高、表达准的一条代表。
- 如果多个高赞想法方向相反但都贴文本，必须作为“外部读者分歧”保留，不要只选更赞同我的一条。
- 与我的想法无关但独立精彩的高赞想法，不要强行解释成“对我的想法的补充”，直接标为“独立精彩外部视角”。
- 外部读者原话可作为短句随对应卡片内联，必须明确标为“外部读者”并说明其关系，不得改写成笼统的“有读者说”。没有直接关系但独立精彩的材料，不强行解释，保留为全局“外部读者精彩高赞想法”候选即可。
- 第二阶段可以在对应条目下保留“外部读者高赞想法分析”，但不得让高赞想法覆盖我的原想法、AI评价、AI修正和AI补充，也不要求每张卡都有外部读者材料。
